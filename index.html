The Gallery: Selected Works and Case Studies

In the 3D gallery room, you can walk through an exhibition of my most significant projects. Here is a detailed breakdown of my work.

Autonomous Parcel Delivery Drone (APDD)

A GPS-guided quadcopter built on the F450 frame with a Pixhawk 2.4.8 flight controller running ArduCopter firmware. Designed to navigate GPS waypoints autonomously and release parcels via an MG90S servo-actuated hook on arrival.

Multi-Modal Cancer Omics Classifier

Fused clinical, genomic, and transcriptomic data into a multi-modal machine learning pipeline to predict cancer outcomes. Achieved a perfect AUC score on the evaluation set and won the **NeoHack 2025** hackathon.

AI Customer Support Agent (AJ Digital Agency, Mardan)

Deployed a production RAG-powered lead generation support agent utilizing n8n, Pinecone vector store, and Gemini embeddings. Structured, debugged, and optimized vector store ingestion pipelines and updated system prompts for high-converting lead capture.

SpreadsheetAppBuilder (SAB)

Open-source engine converting natural-language prompts to spreadsheet-based applications via a JSON blueprint pipeline. Decoupled generation layers from rendering adapters, supporting both LibreOffice (Python/UNO) and Excel (Office.js).
